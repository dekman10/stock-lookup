# Stock Lookup

A Flask web app for looking up stock data and comparing stocks side-by-side. Uses [yfinance](https://github.com/ranaroussi/yfinance) to fetch real-time market data.

## Features

- Look up any stock by ticker symbol
- Compare two stocks side-by-side
- View current price, daily change, 52-week range, market cap, and analyst ratings
- Dark-themed UI with color-coded indicators

## Setup

```bash
pip install -r requirements.txt
python app.py
```

Then open http://localhost:5000 in your browser.
